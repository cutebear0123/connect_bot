# connect_bot
a discord bot connect guild

# installtion
### clone this
### put your token at first line of __main__.py
### install pycord(and uninstall discord.py if you have)
```bash
pip install -U git+https://github.com/Pycord-Development/pycord
```
### cd to connect_bot
### run
```bash
python .
```
### done
